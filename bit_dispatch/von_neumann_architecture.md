# 冯诺依曼体系

> 冯·诺依曼体系结构（Von Neumann Architecture）是一种计算机系统的设计原则，于 1945 年由匈牙利裔美国数学家、物理学家约翰·冯·诺依曼（John von Neumann）首次提出。

# 主要特点

-   计算机可以在同一内存中存储程序和数据，使得程序可以像数据一样被操作、修改和传输
-   采用了二进制表示法，简化了电路设计和实现
-   确定了计算机的五个基本组成部分

1. 输入设备
2. 存储器
3. 控制器
4. 运算器
5. 输出设备
